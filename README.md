## 数据架构 v2.0 — 4 大类数据 + 5 档更新节奏

### 4 大类数据（按优先级）
1. **实时盘口高频** — Level-1五档盘口 / IOPV实时净值 / 分时逐笔成交 / 1分钟K线快照
2. **多周期历史K线** — 5/15/30/60分钟 + 日/周/月线，全部前复权
3. **ETF专属基本面资金** — 总/流通份额 / PCF申赎清单 / 折溢价率 / 融资融券 / 跟踪指数
4. **宏观&板块对照** — 行业板块涨跌/资金流 / 宽基基准 / 大盘情绪 / 行业宏观催化

### 5 档更新节奏
- 交易时段实时 (9:15-15:00, 10-15秒) — 盘口/IOPV/逐笔/1minK线
- 盘后增量 (15:30后, 每日) — 日线/分钟线归档/份额/折溢价/PCF
- 周度 (周五收盘) — 周线/板块周度资金/回测绩效
- 月度 (月末) — 月线/规模变化/宏观指标
- 静态基础 (变更才更新) — ETF代码池/费率/申赎门槛

## 数据说明
- 日/周/月K线、行情、资金流、ETF详情、新闻：腾讯自选股接口（westock-data）
- Level-1盘口/IOPV/逐笔/日内分钟线：需后端实时行情源，当前为收盘快照（标注 `_realtime:false`）
- 每日自动化（WorkBuddy, 8:30）刷新数据并通过 GitHub API 推送
